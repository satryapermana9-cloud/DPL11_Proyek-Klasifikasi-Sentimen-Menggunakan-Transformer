Deskripsi Proyek

Proyek ini bertujuan untuk melakukan klasifikasi sentimen (Positif, Negatif, dan Netral) pada kalimat berbahasa Indonesia menggunakan model Transformer Encoder berbasis PyTorch.
Model dilatih menggunakan dataset berlabel, kemudian digunakan untuk memprediksi sentimen pada data uji yang belum memiliki label.
